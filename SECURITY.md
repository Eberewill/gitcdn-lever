# Security Policy

## Supported Versions

Security fixes are currently provided for the latest code on `main`.

| Version | Supported |
| ------- | --------- |
| main    | yes       |
| older   | no        |

## Reporting a Vulnerability

Please report vulnerabilities privately.

Preferred path:

- Use GitHub private vulnerability reporting / security advisories for this repository.

If private reporting is unavailable:

- Open an issue with minimal detail and request a private contact channel.
- Do not include exploit code or sensitive data in public issues.

## Response Expectations

- Initial triage target: within 5 business days.
- Status updates are shared as triage progresses.
- Confirmed issues are fixed and disclosed responsibly.
